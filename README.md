# falppy-bird-game

# 🐤 Flappy Bird Game (C++ | SFML)

A graphical clone of the classic **Flappy Bird** game built using **C++ and SFML (Simple and Fast Multimedia Library)**. This project demonstrates real-time game development concepts like physics, collision detection, and rendering using SFML.

---

## 🎮 Gameplay

Control the bird and navigate through moving pipes without crashing. The game becomes progressively challenging as you survive longer.

---

## 🚀 Features

- 🎨 Graphical interface using SFML  
- 🐤 Smooth bird movement with gravity physics  
- ⬆️ Jump control using spacebar  
- 🌿 Randomly generated pipes  
- 💥 Collision detection system  
- 🎯 Score tracking  
- ⚡ Real-time rendering (60 FPS)  

---

## 🛠️ Tech Stack

- **C++** – Core programming language  
- **SFML** – Graphics, window handling, and rendering
